# Intuit Build Challenge Simulation

## Features

### 1. Producer-Consumer System
- Bounded buffer
- Thread-safe implementation
- wait() / notifyAll()
- ExecutorService thread pool

### 2. CSV Analytics Engine
- Stream-based aggregation
- Grouping & Summation
- Clean architecture

## Technologies
- Java 17
- Maven
- JUnit 5

## How to Run
- Run PCApplication
- Run AnalyticsApplication
- Run tests using `mvn test`
